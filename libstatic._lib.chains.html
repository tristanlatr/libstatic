def ast_to_gast(node: ast.Module) -> Tuple[gast.Module, Mapping[gast.AST, ast.AST]]: (source) ¶

This function returns a tuple which first element is the gast module and the second element is a mapping from gast nodes to standard library nodes. It should be used with caution since not all nodes have a corespondance. Namely, expr_context nodes and the store Name of ExceptHandler are not present in the mapping.
