class MutableState(State): (source)

Constructor: MutableState(msg)

View In Hierarchy

Class providing modifiers for the `State`.

Among others, it that ensures that modifications of the def-use chains are replicated in the use-def chains as well, as if they form a unique data structure.

def adopt_expression(self, expr: ast.expr, ctx: Scope): (source) ¶

Updates the ancestors and the chains such that this manually constructed expression is usable in the project like other nodes. This can be useful for instance to register unstringed annotations into the state.

Warlus operator inside adopted expression is not supported: the expression should be pure.
